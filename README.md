# pcsp

A WIP wanna be psp emulator 

# Current Status

- Automatically loaded a binary file (minifire demo code)  
- Basic Dear ImGUI window rendering
- Simple memory class
- Debug memory viewer capable of reading from different areas (ram,varm,scratchpad)
- Able to load pbp files (not in psp's memory) and display it's header 
- Able to load embedded psf files from pbp and display it's keys,values
- Basic Dissasembler , with only a few instructions support
- Basic Allegrex Decoder with a few only decoding instructions
